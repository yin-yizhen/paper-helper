# 环境与失败恢复

首次使用时先确认以下能力：网络；可访问 CNKI 的浏览器会话；浏览器自动化工具；写入文件的工作目录；DOCX 创建与渲染工具。

| 状态 | 可以完成 | 不能声称完成 | 恢复方式 |
| --- | --- | --- | --- |
| 无 CNKI 登录 | 题录检索、部分元数据核验 | 全文下载 | 用户在浏览器登录后继续 |
| 验证码出现 | 已完成内容和台账保存 | 验证码后的检索/下载 | 用户手动完成验证码后继续 |
| 无机构权限 | 题录核验与草稿 | 无权限全文已下载/已阅读 | 记录 `no_access`，由用户切换有权限会话 |
| 无浏览器工具 | 选题、结构、待核验草稿 | CNKI 已核验 | 让用户启用浏览器能力或自行提供可核验题录 |
| 无 DOCX 渲染 | 可生成 DOCX（若有生成器） | 已视觉验收 | 在具备渲染能力的 Codex 环境复验 |

绝不要求用户交出密码、Cookie 或验证码。不要尝试绕过登录、付费、验证码或下载限制。
