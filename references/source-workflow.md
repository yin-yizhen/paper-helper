# 来源筛选规则

仅使用能在 CNKI 结果页与详情页定位的中文期刊文章。候选条目至少核对作者、题名、期刊、发表信息、CNKI 详情链接与主题相关性；不要以搜索引擎摘要代替 CNKI 收录证明。

优先顺序为：本学科 CSSCI / 北大核心 / CSCD / 权威专业期刊；不足时补充有完整出版信息的高质量普刊。期刊层级必须由 CNKI 期刊索引页逐刊核验，未核验标记为 `未核验`，不得写成核心。

下载仅限于最终将写入正文且已有文内引文位置的条目。下载状态只能使用 `downloaded`、`catalog_verified`、`waiting_for_login`、`no_access`、`not_available`、`not_attempted`；不能以“已点击”代替“已下载”。

若没有全文，只能依据明确可见的题录与摘要做有限概括，并在台账中如实记录；不能将摘要中的细节表述成全文阅读结论。
