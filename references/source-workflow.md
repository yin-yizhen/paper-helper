# 知网文献核验与下载规则

在需要检索、下载或著录文献时阅读本文件。

## 准入条件

仅采用在知网结果页能定位到的中文期刊论文。逐条记录：作者、题名、刊名、发表日期、卷期页码或网络首发状态、知网详情链接。

优先顺序：本学科 CSSCI / 北大核心 / CSCD / 权威专业期刊；其后为有明确编辑出版信息的高质量普通学术期刊。排除报纸、营销内容、无法确认出版信息的网页和与主题不直接相关的文章。

## 下载处理

只下载最终入选且会在正文引用的文献。下载前确认题录和详情链接一致。出现登录、验证码、学校权限或付费限制时，请用户完成页面操作；不要绕过限制。

记录每篇文献的状态：`已下载`、`仅题录核验`、`用户无权限`、`等待用户登录`。全文不可得时，不将其摘要中的细节写成已经阅读全文所得的结论。

## 引文闭环

每条参考文献至少有一处正文引用；每个关键外部事实、数据、案例过程或他人结论都有已核验来源。未被引用的文献从最终清单删除。

没有正式卷期页码的网络首发文章，按学校要求使用 `[J/OL]` 著录；若学校不接受网络首发文献，则替换为已正式刊出的文章。
