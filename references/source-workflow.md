# 来源筛选与阅读规则

仅使用能在 CNKI 结果页和详情页双重定位的中文期刊文章。默认 6000 字综述配置 15—20 篇最终来源：以近五年研究为主体，同时保留 2—4 篇确有理论或研究史价值的早期文献；不能为凑“新”而全部选同一年网络首发文章。其他字数按论证需要增减，不机械凑数。

优先本学科 CSSCI、北大核心、CSCD 与权威专业期刊；确有必要时可补充高质量普刊，并在 `selection_tier` 中如实区分。标为核心的期刊必须逐刊取得 CNKI 期刊导航证据。

候选条目至少核对作者、题名、期刊、出版状态、CNKI 详情链接和稳定标识。每次换关键词后重新应用并检查筛选条件，防止 CNKI 自动清空来源类别。

锁定最终清单后，必须下载并通读全部最终来源，再建立“观点—全文页码/章节—正文位置”对应关系。没有全文的条目可以留在候选池，但不能进入最终引用清单，也不能据此动笔。下载状态不能以“已点击”代替“本地文件存在”。

若用户要求“直接交付”，上述全文门禁不可降级。CNKI、机构权限或下载失败时硬停止并保留已有证据，恢复后续做；不生成摘要拼接稿或待核验版。
